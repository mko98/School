  1. Wat is de betekenis van TCP?

    2. Transmission Control Protocol

    Het is een protocol dat veel gebruikt wordt voor gegevensoverdracht over netwerkverbindingen op het internet en op computernetwerken.

  2. Wat is de betekenis van IP-adres

    3. Internet Protocol adres

    Het IP-adres is een adres waarmee een netwerkkaart van een host geadresseerd kan worden binnen het IP-model. DIt betekent dat elke computer een nummer krijgt, waarmee die zichtbaar is voor alle andere computers op het internet.

  3. Wat is de betekenis van DNS

    4. Domain Name System

    DNS is is het systeem en netwerkprotocol dat op het Internet gebruikt wordt om namen van computers naar numerieke adressen (IP-adressen) te vertalen en omgekeerd.

  4. at is de betekenis van Router

    5. Met een router leg je een veilig bedraad en draadloos netwerk aan in je huis of op kantoor.

  5. Wat is de betekenis van webserver

    6. Een webserver is een programma dat via een netwerk HTTP-verzoeken ontvangt en documenten naar de client stuurt.

  6. Wat is de betekenis van HTML

    7. HyperText Markup Language

    HTML is een op SGML gebaseerde opmaaktaal voor de specificatie van documenten, voornamelijk bedoeld voor het world wide web. Het is de standaard opmaaktaal voor webpagina's

  7. Wat is de betekenis van HTML tag

    8. Een HTML tag is een opdracht die door een browser kan worden ge√Ønterpreteerd. HTML gebruikt tags om bijvoorbeeld de positie en weergave van elementen op een pagina aan te geven.

Hoe weet een browser wat voor soort bestand door de server wordt geretourneerd?

Een browser weet via de content-type welk soort bestand wordt gestuurd.

Wat is het doel van keywords bij een website? Wie of wat gebruikt deze?

Door de keywords kunnen zoekmachines de inhoud van een pagina vaststellen. Het wordt gebruikt door zoekmachines

Wat is het doel van een description? Waar zal deze, wanneer je je site publiceert, te zien zijn?

Door een description kan de lezer voordat ze de site op gaan door middel van een samenvatting lezen waar de site over gaat.

Klik in je browser met je rechter muisknop op je pagina, en kies Pagina weergeven (of Ctrl+U). Wat zie je?

Door crtl+U te gebruiken kun je de broncode van de website zien waar je op dat moment bent.

Klik op dezelfde wijze, en selecteer Element inspecteren (of Ctrl+Shift+I). Beweeg met je muis over het scherm. Wat zie je? Wat is het nut hiervan?

De functie Element Inspecteren laat je de HTML en Stylesheet (CSS) van de webpagina zien, die je op dat moment bekijkt. Je kunt zien over welk onderdeel het gaat door je muis te bewegen.

Maak een afbeelding van je browser wanneer je een element inspecteert, kopieer deze in Paint en sla deze op als JPG bestand in je images map. Voeg deze afbeelding vervolgens op je pagina toe.
